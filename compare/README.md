# matplotlib side-by-side comparison

Generates matching figures with **plotine** and **matplotlib** for visual review.

```bash
# from repo root
cargo run -p plotine --example matplotlib_compare
python scripts/matplotlib_compare.py   # pip install matplotlib numpy
```

Then open the local viewer:

```bash
# Windows
start compare/index.html
```

PNG outputs are gitignored; `index.html` is regenerated by the Python script.
