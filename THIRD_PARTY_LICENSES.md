# Third-party licenses

plotine embeds or depends on the following third-party assets.

## Embedded resources

### DejaVu Sans fonts

**Files:** `crates/plotine-text/fonts/DejaVuSans.ttf`, `DejaVuSans-Oblique.ttf`

DejaVu fonts are derived from Bitstream Vera fonts. DejaVu changes are in the
public domain. The full license text (Bitstream Vera + Arev + TeX Gyre DJV Math)
is at [`crates/plotine-text/fonts/LICENSE`](crates/plotine-text/fonts/LICENSE).

Redistribution requires including the copyright and trademark notices from that
file.

### Natural Earth 110m coastline

**File:** `crates/plotine/src/geo/data/coastline.bin`

Derived from [Natural Earth](https://www.naturalearthdata.com/) 1:110m
coastline data. Natural Earth data is **public domain** — no license
restrictions apply.

Generated by `scripts/build_coastline.py` from the upstream GeoJSON.

## Rust dependency licenses

All Rust crate dependencies (direct and transitive) use permissive open-source
licenses. The most common are:

- MIT
- Apache-2.0
- MIT OR Apache-2.0
- BSD-2-Clause / BSD-3-Clause
- ISC
- Unicode-3.0
- Zlib
- BSL-1.0

No GPL, AGPL, or other strong copyleft dependencies are present in the
dependency tree. Use `cargo deny check licenses` (with the provided
[`deny.toml`](deny.toml)) to verify.

## Optional system tools

Some optional features invoke external system tools. These are **not** bundled
with plotine; users install them separately under their own licenses:

| Feature | Tool | License |
|---------|------|---------|
| `eps` | Ghostscript (`gs`) | AGPL-3.0 (invoked as subprocess, not linked) |
| `mp4` | ffmpeg | LGPL-2.1+ / GPL-2.0+ (invoked as subprocess) |
| `latex` | TeX + dvipng | Various (invoked as subprocess) |

These tools are executed as separate processes and are not compiled into or
distributed with plotine.
